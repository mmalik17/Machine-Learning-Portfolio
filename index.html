Improving Marketing Campaign Performance
by Predicting Customer Responses

The marketing campaign carried out by a retail company was less effective, as previous campaign response only reached 14.91%. As a results, the company profitability is not optimal. In this project, the machine learning model predicts which customer that will response to the campaign, and which one will not response.

After the model give the result, we will compare the profitability of the company before and after applying machine learning model. The model is also capable to rank the feature importances to identify which features have big impact, and which features that have small impact

Objectives:
1. Create a model that could predict which customer will response to the campaign and which customer will not respond
2. Compare the business metric in the company before and after applying the model
3. Propose business reccomendation about strategy to improve marketing campaign performance

Challenge:
1. There are many categorical features that need to be converted by perform label encoding and one-hot encoding
2. Make sure that the data is clean, which means free from missing value, duplicated value, and outlier.

Identifying Spam Email Text with
Natural Language Processing

Spam email is an unwanted message that come to our email account, mostly from mass broadcast. Usually, spam messages have certain simillar pattern, such as offering discount, offering free product, etc. The opposite of the spam email is ham email.

In this project, the machine learning model predicts which email considered as spam or ham. The data is splitted into two section, 80% training data and 20% test data. The future purpose of this machine learning is selected the spam email to spam folder, so the primary folder is cleaned from spam or unwanted email.

Objectives:
Create a model that could differentiate spam and ham email with natural language processing (NLP) technique

Challenge:
The model should have accuracy more than 85%

Image Classification with
Deep Learning

In this project, there are 1000 dog images, 1000 cat images, and 1000 panda images. Every image is labelled with the suitable animal name. The machine learning paired image data and label data. Then, it analyzes the characteristic of dog, cat, and panda. The data is splitted into two section, 80% training data and 20% test data.

Objectives:
Create a model that could differentiate the animal name inside the image

Challenge:
The data source is not from the regular table, but from the folder/directory. The model should have accuracy more than 85%
